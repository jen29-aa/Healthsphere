Overview

Healthsphere is an intelligent healthcare support system that predicts possible diseases based on user-selected symptoms using Machine Learning (Naïve Bayes classifier) and recommends the appropriate medical department or doctor.
The system also provides precautionary guidance, health articles, and visual charts to help users better understand health trends.

 Key Features
 1. Symptom-Based Disease Prediction
      --Users select symptoms from a predefined list.
      --ML model (Naïve Bayes) predicts the most probable disease.
      --Fast response time (< 3 seconds).
2. Doctor Recommendation System
      --Maps predicted disease to the correct medical specialty./n
     --Suggests relevant departments (e.g., Cardiology, Neurology).
      --Precautions Display
      --Shows general precautionary steps based on predicted disease.
      --Helps users manage symptoms before consulting a doctor.
 3. Graphical Insights
      --Generates charts showing the most frequently predicted diseases.
      --Helps users understand health trends.
4. Health Articles
